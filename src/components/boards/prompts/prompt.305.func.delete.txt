            1) 라이브러리조건
                - API: 이미 설치되어있는 @tanstack/react-query를 사용할 것.

            2) 삭제 기능을 적용할 것.
                1. GalleryCard 컴포넌트
                    1-1) 게시물 카드에 호버 시 삭제 버튼이 표시되도록 할 것.
                    1-2) 삭제 버튼은 lucide-react의 Trash2 아이콘을 사용할 것.
                    1-3) 삭제 버튼은 호버되지 않았을 때는 보이지 않을 것.
                    1-4) 삭제 버튼 클릭 시 이벤트가 부모 컴포넌트로 전달되도록 할 것.

                2. 삭제 확인 다이얼로그
                    2-1) 삭제 버튼 클릭 시 "게시물 삭제하겠냐" 확인 다이얼로그를 표시할 것.
                    2-2) 확인 다이얼로그는 Modal 컴포넌트를 사용할 것.
                    2-3) 확인 다이얼로그에는 "취소"와 "삭제" 버튼이 있을 것.
                    2-4) "취소" 버튼 클릭 시 다이얼로그를 닫고 아무 동작도 하지 않을 것.
                    2-5) "삭제" 버튼 클릭 시 게시물 삭제 API를 호출할 것.

                3. deleteBoard API
                    3-1) API명: deleteBoard
                    3-2) 요청데이터:
                        boardId: ID!
                    3-3) 응답데이터: Boolean
                    3-4) 삭제 성공 시 게시물 목록을 refetch하여 업데이트할 것.
                    3-5) 삭제 실패 시 에러 메시지를 표시할 것.

                4. 삭제 후 동작
                    4-1) 삭제 성공 시 해당 게시물이 목록에서 제거될 것.

                    4-3) 삭제 성공 시 성공 메시지를 표시할 것.
